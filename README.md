            <p>This project uses OpenCV to detect and count vehicles passing through a virtual counting line in a video. It employs background subtraction techniques to identify moving objects and tracks them as they cross a predefined boundary.</p>

            <h2>‚ú® Features</h2>
            <ul class="feature-list">
                <li>Real-time vehicle detection using background subtraction</li>
                <li>Automatic counting when vehicles cross the counting line</li>
                <li>Visual feedback with bounding boxes and center points</li>
                <li>Configurable detection parameters</li>
                <li>Counter display on video feed</li>
            </ul>

            <h2>üîç How It Works</h2>
            <ol>
                <li><strong>Background Subtraction:</strong> Uses MOG (Mixture of Gaussians) algorithm to separate foreground (vehicles) from background</li>
                <li><strong>Preprocessing:</strong> Applies Gaussian blur and morphological operations to reduce noise</li>
                <li><strong>Contour Detection:</strong> Identifies vehicle shapes using contour detection</li>
                <li><strong>Center Tracking:</strong> Calculates center point of each detected vehicle</li>
                <li><strong>Line Crossing Detection:</strong> Counts vehicles when their center crosses the counting line</li>
            </ol>
